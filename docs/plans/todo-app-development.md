# TODOアプリケーション開発計画書

## プロジェクト概要

React Router v7 + Cloudflare Workers + D1 + shadcn/ui を使用したTODOアプリケーションの開発。
既存の認証基盤を活用し、ユーザーごとのタスク管理機能を提供する。

## 機能要件

### タスク管理機能

- タスクの作成、編集、削除
- ステータス別の3エリア表示（未着手、着手中、完了）
- ドラッグ&ドロップでステータス変更
- ダイアログでタスク作成・編集

### タスク仕様

- タイトル（必須）
- 内容（必須）
- 期限（オプション）
- 優先度（high/medium/low）
- ステータス（not-started/doing/done）
- 作成者・更新者情報

## 技術要件

### 既存システムとの統合

- Better-Auth認証システムとの連携
- 既存のD1データベース構成の活用
- shadcn/ui コンポーネントライブラリの使用
- React Router v7のSSR機能の活用

### データベース設計

- tasksテーブルの追加
- 既存のuserテーブルとのリレーション
- インデックス設計の最適化

### フロントエンド設計

- ドラッグ&ドロップライブラリの統合
- レスポンシブデザインの実装
- リアルタイム更新の検討
